## Grocery Store Web App 
A grocery store web app written in python, using the flask framework, sqlite database and bootstrap css.
      
    
To setup the project, first navigate to the project folder with your command prompt and then run
  
For Windows,
```
python -m venv venv
venv\Scripts\activate
```
For Linux,
```
apt update
apt install -y virtualenv
virtualenv venv
source ./venv/bin/activate   
```

Install dependencies
```
pip3 install -r requirements.txt
```
and then to run the application
```
python3 app.py
```
to add an admin, modify the create_manager function and make changes for username and password
